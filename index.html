01

Roche Scientist Assistant · MBDS Capstone ↗

Capstone project for Roche: an AI assistant that gives scientists a single place to get work done. A retrieval-augmented (RAG) engine answers questions from internal SOPs and lab procedures, with every answer grounded in and citing its source document, and low-confidence answers flagged rather than fabricated. Beyond Q&A, the assistant auto-triages and files IT incidents into ServiceNow (setting category, urgency and routing to the right team), captures feedback with automatic sentiment analysis, and works across English, German, French and Italian with browser voice input. Built as a FastAPI and React application using Google Gemini embeddings and LLM over a ChromaDB vector store, with a Google Drive sync bridge keeping the knowledge base live.

RAG Capstone Agentic AI FastAPI React Gemini ChromaDB ServiceNow
02

EV Charging Network Design: IEU Sustainability Datathon (Finalist) ↗

Built an end-to-end EV charging network analysis for the IE University x Iberdrola Sustainability Datathon, designing Spain's 2027 interurban charging infrastructure using real traffic, road, and grid data. Ingested and cleaned multi-source datasets (traffic flows, road segments, existing fast chargers, and DSO grid capacity), then projected 2027 EV demand and modeled long-distance EV trips on AP/A/N corridors. Optimized station placement with AFIR spacing rules and matched proposed sites to substation capacity, delivering data-backed outputs: a proposed station network, a friction-point heatmap, and an interactive visualization. The project combined geospatial analysis, demand forecasting, and infrastructure optimization to produce a practical EV rollout plan aligned with regulatory spacing and grid constraints. Recognized as a finalist among all competing teams.

03

Proactive Security Orchestration Agent (Winner 2025)

Built an agentic AI system on top of BMC Helix that autonomously monitors IT infrastructure, detects anomalies, and orchestrates incident responses without human intervention. The agent continuously ingests telemetry data across services, correlates signals to identify potential threats before they escalate, and triggers automated remediation workflows (ticket creation, service restarts, escalation routing). Designed around a multi-agent architecture where a coordinator agent delegates to specialized sub-agents for log analysis, risk scoring, and action execution. Won first place at IE University, Madrid (2025).
